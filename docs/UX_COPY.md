# Evergreen Market — UX Copy Rules

## Language

All public customer-facing UI copy should be Ukrainian.

Do not add new Russian text to public storefront UI unless explicitly requested.

## Tone

Use simple, warm, everyday language.

Good tone:

- clear;
- calm;
- local;
- honest;
- human.

Avoid:

- legal wording;
- corporate wording;
- technical supplier jargon;
- complicated B2B terms.

## Preferred phrases

```txt
Як це працює?
```

```txt
Забрати в кавʼярні
```

```txt
Ми закуповуємо товари для кавʼярні напряму у постачальників.
```

```txt
Частину цих товарів ви можете замовити для себе.
```

```txt
Коли замовлення буде готове, ми повідомимо вас.
```

```txt
Зайшли за кавою — і забрали своє замовлення.
```

## Avoid phrases

Avoid using these in public UI:

```txt
B2B-постачальники
```

```txt
комерційна модель
```

```txt
ми є посередником
```

```txt
оптимізація логістики
```

```txt
точка видачі
```

## Minimum order explanation

Good:

```txt
Постачальники зазвичай працюють із мінімальними сумами замовлення. Ми показуємо ці умови одразу, щоб ви розуміли, як працює замовлення.
```

Short:

```txt
Для цього постачальника діє мінімальна сума замовлення.
```

## Delivery explanation

Since delivery is not active yet, use future wording only.

Good:

```txt
У майбутньому ми хочемо додати доставку поруч із кавʼярнею.
```

Bad:

```txt
Ми доставимо замовлення додому.
```
