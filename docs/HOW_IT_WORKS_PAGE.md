# Page Specification — “Як це працює?”

## Purpose

Create an explanatory public page for Evergreen Market.

The page should help customers understand:

- where products come from;
- why prices may be better than in regular retail;
- why minimum order exists;
- how pickup works;
- that future local delivery may appear later;
- why Evergreen is doing this.

## Route

Preferred route:

```txt
/how-it-works
```

Navigation label:

```txt
Як це працює?
```

H1:

```txt
Як працює Evergreen Market?
```

## Implementation requirements

- Add route `/how-it-works`.
- Add navigation link `Як це працює?`.
- Add footer link if footer exists.
- Keep route changes in sync with `src/utils/routes.js` if used.
- Add page meta in `src/utils/pageMeta.js` if used.
- Add route to backend sitemap output in `server/seoRoutes.cjs` if static routes are listed there.
- Use existing layout and styling patterns.
- Customer-facing text must be Ukrainian.
- Mobile-first.
- Do not redesign the whole site.

## Suggested page structure

Title:

```txt
Як працює Evergreen Market?
```

Subtitle:

```txt

```

### 2. Звідки беруться товари?

Text:

```txt
Кавʼярня щодня працює з молоком, рослинним молоком, сиропами, чаєм, кавою, напоями та іншими товарами. Частину цієї продукції ми закуповуємо напряму у постачальників.

Коли кавʼярня купує напряму, вона не переплачує за роздрібну націнку магазину. Саме тому частину цих товарів ми можемо пропонувати нашим гостям у зручному форматі замовлення.
```

### 3. Чому це може бути вигідніше?

Text:

```txt
Багато товарів, які ми використовуємо в кавʼярні, люди купують і додому: молоко, рослинне молоко, чай, кава, сиропи, напої, снеки та солодощі.

У звичайному магазині до ціни часто додається роздрібна націнка. Evergreen Market дає можливість замовляти частину товарів через кавʼярню — простіше, ближче і часто вигідніше.
```

Speech bubble:

```txt
Ви не купуєте величезну партію, але все одно отримуєте більш вигідні умови.
```

### 4. Чому існує мінімальне замовлення?

Text:

```txt
Постачальники зазвичай працюють не з окремими маленькими покупками, а з мінімальними сумами замовлення.

Ми беремо на себе складну частину — роботу з постачальниками. Вам залишається просто обрати потрібні товари, оформити замовлення і забрати його в Evergreen.
```

Visual idea:

```txt
Постачальник → Evergreen → Ваше замовлення
```

### 10. FAQ

#### Чи можна замовити один товар?

```txt
Так, якщо товар є в наявності в кавʼярні. Для товарів під замовлення може діяти мінімальна сума.
```

#### Чому для деяких товарів є мінімальне замовлення?

```txt
Тому що постачальники зазвичай працюють із мінімальними сумами закупівлі. Ми намагаємося зробити ці умови простішими для звичайних покупців.
```

#### Коли я зможу забрати замовлення?

```txt
Ми повідомимо вас після підтвердження замовлення. Якщо товар є в наявності, його можна буде забрати швидше.
```

#### Де забирати замовлення?

```txt
У кавʼярні Evergreen.
```

#### Чи буде доставка?

```txt
Поки що основний формат — самовивіз із кавʼярні. У майбутньому ми хочемо додати безкоштовну доставку поруч із Evergreen.
```

#### Це дешевше, ніж у магазині?

```txt
Часто так, тому що ми працюємо з постачальниками напряму. Але ціни можуть залежати від конкретного товару, постачальника і умов закупівлі.
```
